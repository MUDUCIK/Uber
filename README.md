## System Requirements

- [git](https://git-scm.com/)
- [NodeJS](https://nodejs.org/en/)

<br/>

## Run Locally

Clone the project

```bash
  git clone https://github.com/mudaston/Uber.git
```

Go to the project directory

```bash
  cd Uber
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  gulp
```

Open in a browser

```bash
  http://localhost:3000
```
